Human-friendly

  • Rich tables and formatted output
  • Full purchase flow: search вҶ’ cart вҶ’ checkout
  • Price comparison across stores

Agent-friendly

  • REST API вҖ” FastAPI on localhost
  • MCP Tools вҖ” 9 tools over stdio
  • JSON parseable output
  • Autonomous purchase workflows
