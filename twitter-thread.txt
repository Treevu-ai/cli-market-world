TWEET 4 — PARA HUMANOS Y AGENTES

Esto no es solo para humanos.

Human-friendly: comandos, tablas, flujo de compra
Agent-friendly: REST API, MCP tools, JSON

market --json → parseable por cualquier LLM
market_mcp.py → 9 tools MCP para DeepSeek TUI, Claude, etc.

Stripe transformó pagos en APIs. Nosotros transformamos supermercados.
