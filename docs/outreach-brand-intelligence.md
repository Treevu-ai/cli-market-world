## NOTAS DE EJECUCIÓN

### Secuencia recomendada

**Semana 1:**
- Contactar las 5 agencias (email o LI DM)
- Preparar el demo con datos reales: Laive vs Gloria (lácteos) + Don Vittorio vs Molitalia (pastas)

**Semana 2:**
- Seguimiento agencias que no respondieron
- Contactar 3 marcas: Molitalia, Laive, Florida

**Semana 3:**
- Cierre agencias
- Seguimiento marcas
- Si hay interesado: onboarding del piloto (30 minutos, manual)

**Semana 4:**
- Cierre marcas
- Inicio piloto del primer cliente
- Primer reporte PDF al final del mes

### Demo deck (para llamada de 20 min)

1. **Min 0-2:** El problema — pantalla con tabla Don Vittorio (datos reales de hoy)
2. **Min 2-8:** El sistema — búsqueda en vivo de un SKU de la marca del prospecto
3. **Min 8-14:** Las 3 vistas — Mis SKUs / Competidores / Promo Tracker (mockup o dashboard real)
4. **Min 14-18:** El modelo — piloto 30 días gratis, $500/mes después
5. **Min 18-20:** Siguiente paso — onboarding esta semana vs "lo consulto con el equipo"

### Reglas de personalización (no saltear)

- **Siempre usar datos reales del día** — no datos genéricos. Consultar la API antes de cada
  envío batch.
- **Mencionar el competidor específico** — el anzuelo es el competidor, no el propio producto.
- **Una sola pregunta de cierre** — no ofrecer múltiples opciones. Siempre: "¿20 minutos esta
  semana?"
- **No mencionar precio en el primer email** — el precio viene en la conversación. El primer
  email solo genera respuesta.
